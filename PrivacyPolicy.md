# **Privacy Policy — Pepper Find**

**Last updated:** *02/12/2025*

Pepper Find (“the App”) is provided by **Kabindas** (“we”, “us”).
This Privacy Policy explains how user information is handled when using the App.

---

## **1. Information We Collect**

Pepper Find **does not collect, store, or share any personal data** from users.

The App only **reads a public JSON file** hosted on GitHub to display information inside the app.
No user information is transmitted to us or stored by the App.

---

## **2. Third-Party Services**

The App uses the following external service:

### **GitHub (Content Hosting Only)**

The App loads a JSON file stored on GitHub.
GitHub may collect standard technical information (such as IP address or device type) when loading the file, according to their own Privacy Policy.

We do **not** send any user data to GitHub — users only download a static file that the App uses.

---

## **3. Children’s Privacy**

Pepper Find is suitable for **general audiences, including children**.
Because the App does not collect personal information, it complies with child-protection requirements, including COPPA and the Google Play Families Policy.

---

## **4. Data Security**

Since the App does not collect or store personal information, no personal data is at risk.
The App only reads public data and performs no user tracking or analytics.

---

## **5. Changes to This Privacy Policy**

We may update this Privacy Policy if the App’s behavior or third-party services change.
Any updates will be posted on this page.
